CSS REFERENCE CHEATSHEET

CSS Implementations

Type/Name Description
Inline Uses a style attribute inside HTML elements
Internal Uses a <style> element in the <head> section
External Uses a <link> element to link to an external CSS file

CSS Properties

Property Example Description
color color: blue; Changes foreground color of selector, either in hex, RGB or built in colors!
background-color background-color: yellow; Changes background color of selector, either in hex, RGB or built in colors!
font-family font-family: Arial; Change the typeface/font of text. Can use web safe fonts, installed fonts (@font-face) or linked fonts from the web
font-weight font-weight: bold; To change the weight of the font i.e Light, Bold, Bolder, 300 etc.
border border: 1px solid red; Implements/specifies the border around a selector, shorthand to include weight, style, color of border
padding padding: 50px; An element's padding is the space between its content and its border.
margin margin: auto; The margin property sets the margins for an element, and is a shorthand property for the following properties: margin-top, margin-right, margin-bottom, margin-left
